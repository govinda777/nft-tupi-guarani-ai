# nft-tupi-guarani-ai
Mais um projeto de nft, porem com o tema tupi guarani.  Por trás existe uma comunidade onde os detentores da NFT tem acesso a uma área privada do site.  Para quem tem um item de uma coleção, irá todos os dias receber um outro item mintado por uma AI.  Esse protocolo é dinamico e flexivel, contamos com outras ferramentas para criar a nft.
